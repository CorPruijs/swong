Swong

Swong is an open-source Pong clone written in Swift, Apple's new programming language.

It's built to test Swift and some of it's features like optionals.

Features

  • Basic pong gameplay
  • Smooth animations and effects
  • Highly customizable
  • Open-source
  • Debug menu that shows a lot of info about the game
