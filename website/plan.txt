Plan van aanpak
===============

Doel van de website
-------------------
Het doel van de website is om een Promotie website te worden voor Swong, mijn Pong clone geschreven in Swift
Dit lijkt me erg nuttig omdat ik later misschien wel meer van dit soort websites moet bouwen.

Informatie die ik ga aanbieden
------------------------------
Alles dat te maken heeft met de game, zoals:
- Screenshots
- Verhaaltje over wat het is
- Feature lijst
- "App Store knop" waarmee je hem kan downloaden (die knop wordt nep, hij is niet echt verkrijgbaar)
- Een knop naar de publieke Github repository zodat andere mensen de source code kunnen bekijken / het project kunnen forken.
- tabel met systeemvereisten
- een "easter egg", waarschijnlijk een kleine JavaScript game.
- eventueel gameplay video, waarschijnlijk niet.

Hoe de site er ongeveer uit gaat zien
-------------------------------------
het wordt een lange pagina waar je doorheen kan scrollen en die alles verteld, dit werkt namleijk een stuk beter dan allemaal losse secties van je website, verbeterd de gebruikers ervaring en werkt ook goed in mobiele browsers.
Om ervoor te zorgen dat je nog wel snel naar andere secties (pagina's) kunt gaan, maak ik een header die aan de bovenkant van de pagina blijft staan, met knoppen die snel naar een ander gedeelte gaan van de website.
